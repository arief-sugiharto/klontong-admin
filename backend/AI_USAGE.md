# AI Usage Statement

## Overview

Project ini dikembangkan dengan bantuan Artificial Intelligence (ChatGPT) sebagai coding assistant.

AI digunakan untuk membantu mempercepat proses pembelajaran, debugging, penyusunan dokumentasi, dan memberikan contoh implementasi.

## AI Contributions

AI digunakan untuk:

- Menjelaskan konsep NestJS
- Menjelaskan penggunaan Prisma ORM
- Membantu implementasi CRUD
- Membantu validasi DTO
- Membantu debugging error
- Membantu integrasi Frontend dan Backend
- Membantu menyusun dokumentasi project

## Developer Responsibilities

Seluruh proses berikut dilakukan secara mandiri oleh developer:

- Memahami requirement skill test
- Menjalankan project
- Melakukan konfigurasi PostgreSQL
- Melakukan konfigurasi Prisma
- Menjalankan Migration
- Testing menggunakan Thunder Client
- Memperbaiki error yang muncul
- Menjalankan Frontend Nuxt
- Melakukan integrasi API
- Verifikasi seluruh endpoint berjalan dengan baik

## Testing

Project telah diuji menggunakan:

- Thunder Client
- Browser
- PostgreSQL

Semua endpoint CRUD berhasil berjalan sesuai requirement.

## Statement

AI digunakan sebagai productivity tool dan learning assistant, bukan sebagai pengganti proses analisis, pengujian, maupun pengambilan keputusan teknis.

Seluruh hasil implementasi telah diverifikasi dan diuji sebelum disubmit.
