# Klontong Admin

Technical Test - Klontong Admin

## Tech Stack

- Backend: NestJS
- Frontend: Nuxt 3
- Database: PostgreSQL
- ORM: Prisma

## Project Structure

```
klontong-admin/
├── backend/
└── frontend/
```

## Backend

```bash
cd backend
npm install
cp .env.example .env
npm run start:dev
```

Default:
http://localhost:3000

## Frontend

```bash
cd frontend
npm install
npm run dev
```

Default:
http://localhost:3001

## Notes

- Environment variables are intentionally excluded from this repository.
- Create your own `.env` using `.env.example`.
